# CLAUDE.md

## プロジェクト概要

架空のコーヒーショップ「CAFÉ MORI」のランディングページ。

## ファイル構成ルール

- HTML は `index.html` にまとめる
- CSS は `style.css` にまとめる（別ファイルに分割しない）
- JavaScript は `script.js` にまとめる（別ファイルに分割しない）

## デザイン方針

- シンプルでモダンなデザインを維持する
- 既存の CSS 変数（`--color-*`、`--font-*` など）を再利用する
- 新しいカラーやフォントを追加する場合は `:root` の変数として定義する

## コーディング規約

- コメントは日本語で書く
- クラス名は BEM 記法（`block__element--modifier`）に従う
- インデントはスペース 2 つ
